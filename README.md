# Agentic Engineering Workflow Automation

Frontend application for agentic FEA (Finite Element Analysis) workflow automation, built with Next.js and featuring interactive 3D model visualization.

**Backend:** [agentic-engineering-workflow-automation](https://github.com/Saasvidu/agentic-engineering-workflow-automation)

## Environment Variables

For environment variable configuration, please refer to the [backend repository](https://github.com/Saasvidu/agentic-engineering-workflow-automation). The MCP server URL defaults to `http://localhost:8000`.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

The application will be available at `http://localhost:3000`.
